# 🎓 School Management System (Python CLI)

A console-based School Management System built with Python. This project demonstrates core programming concepts such as modular design, data handling, and user interaction.

## 🚀 Features
- Student Management (Add, Search, Update, Delete)
- Teacher Management
- Course Management
- Student Enrollment System
- Reports (Top Students, Summary, Course Capacity)

## 🛠 Technologies Used
- Python
- Dataclasses
- CLI (Command Line Interface)

## 📌 How to Run
```bash
python school_management.py
